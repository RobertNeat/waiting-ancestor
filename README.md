## What project do?
___
Simple program forks itself into parent-child configuration, then parent waits for child to end its execution.

![terminal](execute.png)

## How to use?
___
Just compile and run:
```
gcc -o waiting-ancestor waiting-ancestor.c 
./waiting-ancestor 
```
